# Gestione Zone

**Gruppo:** Battistini, Benedetti, Tassi  

**Progetto:** 3ElleOrienta 

**Responsabilità:** CRUD delle Zone + Pagina statica Dashboard Admin

---

##  Descrizione

Questo modulo gestisce le **zone geografiche** 
(es. Jesi, Fabriano, Cingoli).  
Comprende:
- Una **pagina frontend** per la dashboard dell'amministratore che permette di visualizzare, aggiungere, modificare ed eliminare le zone.
- Un **backend** con le operazioni CRUD complete sulle zone.

---

## Come avviare il progetto

### Prerequisiti
- UniserverZ per il server

### Installazione
```bash
git clone https://github.com/AndreaBatt2006/3elleOrienta-Zones
cd 3elleOrienta-Zones

```

---

## Pagina Dashboard Admin

La pagina `frontend/pages/zone.html` permette all'amministratore di:
- Visualizzare l'elenco delle zone presenti nel database
- Aggiungere una nuova zona tramite form
- Modificare il nome di una zona esistente
- Eliminare una zona
